# 資訊網路工程系排課系統

這是一個純前端（HTML/CSS/JavaScript）的排課系統，適合系辦或課務人員快速建立與檢視每週課表。

## 功能

- 登入畫面（老師登入 / 學生登入）
- 登入驗證碼（可重新產生）
- 老師登入後三階段後台（填報、課綱管理、選課監控）
- 學生登入後四大區塊（儀表板、搜尋篩選、購物車志願序、5x10 模擬課表）
- 課程新增、編輯、刪除
- 課程類型管理（必修 / 選修）
- 即時衝堂檢查（同一天時間重疊會阻擋）
- 每週課表視圖（週一到週五，08:00~20:00）
- 課程數、學分、教師、教室統計
- 本地儲存（LocalStorage）
- JSON 匯出 / 匯入
- 一鍵載入範例資料

## 使用方式

1. 直接用瀏覽器開啟 `index.html`。
2. 先使用登入畫面登入。
3. 在「課程編輯」填寫課程資料後，點擊「新增課程」。
4. 在「課程清單」可編輯或刪除課程。
5. 使用「匯出 JSON」備份資料；可透過「匯入 JSON」還原。

- 學生：`D1124182008` / `Q224620840`

> 注意：目前是前端示範版登入，帳號資料寫在程式中，不適合正式上線環境。

## 老師端重點

- 填報階段：可設定擋排時段、教室需求、合授比例（A+B=100）
- 擋排限制：單日擋排不可超過 4 節、不可連續 6 節、週五下午不可全擋
- 管理階段：提供 1~18 週課綱編輯、核心能力勾選、成績配比（總和需 100）
- 開課後：可監控已選人數/容量、查看學生清單、產生授權碼

## JSON 格式範例

```json
[
  {
    "code": "INE301",
    "name": "網路程式設計",
    "teacher": "劉老師",
    "credits": 3,
    "type": "required",
    "day": "Wed",
    "start": "10:00",
    "end": "12:00",
    "room": "E301"
  }
]
```

`day` 只能是：`Mon`, `Tue`, `Wed`, `Thu`, `Fri`。
`type` 只能是：`required`（必修）或 `elective`（選修）。
